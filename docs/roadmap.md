# Product Roadmap

## MVP (0-6 weeks)
- CI trigger and script orchestration
- Static + AI pipeline skeleton
- Unified issue model and report generation
- PR summary comment posting

## v1 (6-12 weeks)
- Real integrations: GitHub/GitLab API, flake8/eslint/mypy/bandit
- Rule config UI/CLI enhancements
- Better deduplication and severity calibration
- Team-level metrics persistence

## v2 (12+ weeks)
- Dashboard (trend analytics, developer experience insights)
- Async distributed execution for large PRs
- Feedback loop for model prompt/rule tuning
- Enterprise features (RBAC, SSO, audit exports)
